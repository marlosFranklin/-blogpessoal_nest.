# 📝 Blog Pessoal API

API RESTful desenvolvida para gerenciamento de um **Blog Pessoal**, permitindo o cadastro de usuários, autenticação segura via JWT e gerenciamento completo de temas e postagens.

O projeto foi construído seguindo boas práticas de arquitetura backend utilizando **NestJS**, com foco em organização modular, segurança e testabilidade.

---

## 🚀 Funcionalidades

- 🔐 Autenticação de usuários com JWT
- 👤 CRUD completo de Usuários
- 🏷️ CRUD completo de Temas
- 📝 CRUD completo de Postagens
- 🧪 Testes E2E para rotas de autenticação
- 📄 Documentação interativa com Swagger
- ☁️ Deploy em ambiente cloud (Render)
